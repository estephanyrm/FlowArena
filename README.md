# FlowArena

Sistema web para la gestión de eventos y venta de boletos desarrollado con Ruby on Rails.

---

## Descripción

FlowArena permite administrar eventos, zonas, boletos y compras, incluyendo gestión de usuarios, pagos y reportes desde un panel administrativo.

---

## Funcionalidades

- Registro e inicio de sesión de usuarios (Devise)
- Panel de administración
- Gestión de eventos
- Gestión de zonas
- Venta de boletos
- Procesamiento de pagos
- Generación de reportes

---

## Tecnologías utilizadas

- Ruby on Rails
- PostgreSQL 
- Tailwind CSS
- Flowbite
- JavaScript (Stimulus)

---

## Instalación

```bash
git clone <URL_DEL_REPOSITORIO>
cd flowarena
bundle install
yarn install
rails db:setup
rails server
```
